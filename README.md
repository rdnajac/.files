# My dotfiles

My dotfiles repository, managed with [chezmoi](https://chezmoi.io/).

```sh
sh -c "$(curl -fsLS get.chezmoi.io)" -- -b "$HOME/.local/bin" init --apply https://github.com/rdnajac/.files.git
```
