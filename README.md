# .files

My dotfiles repository, managed with [chezmoi](https://chezmoi.io/).

## Install script

Installs chezmoi binary to `~/.local/bin`
```sh
sh -c "$(curl -fsLS get.chezmoi.io/lb)" -- init --apply https://github.com/rdnajac/.files.git
```
